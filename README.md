Blog Project

A simple PHP Blog project built with MySQL and Bootstrap.  
It supports English and Arabic languages, with user login and logout functionality.

Features

- Multilingual (Arabic / English)
- User authentication (login & logout)
- Add, edit, and delete posts
- Responsive design using Bootstrap

Built With

- PHP
- MySQL
- Bootstrap 5
